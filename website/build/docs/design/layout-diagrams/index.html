Layout & Diagram Set

Required diagrams and conventions for Light City

Required Diagram Set (Stage 1)​

  • City Plan (Top-Down): Concentric rings, radial spokes, ~12 communospheres
  • Energy Grid: Nodes (spire, spheres, temples), pathways, flow direction
  • Central Spire Cross Section: Structural zones, energy/resonance layers
  • Communosphere Interior Concept: Levels, private/common spaces, resonance hardware
  • Sacred Geometry Overlay: Flower of Life / Seed / Golden sections on city plan

File Conventions​

  • Location: assets/images/diagrams/
  • Naming: diagram-<topic>-v1.png (and source files if applicable)
  • Export: PNG or SVG (prefer SVG for line work)
  • Include legend and scale where possible

Style Conventions​

  • Line: Deep Space Blue (#0B2447) at 70% opacity
  • Highlights: Celestial Blue; Energy flows in Solar Gold → White Gold
  • Background: Mist Gray or transparent
  • Labels: Small caps, muted gray, leader lines at 1px

Versioning​

  • Increment versions: v1, v2 as you iterate
  • Keep sources in assets/images/diagrams/src/ (Figma, .blend, etc.)

Delivery Checklist​

  • Exported PNG/SVG
  • Source file saved
  • Linked from relevant docs (architecture.md, energy-systems.md)
  • Added to assets/images/diagrams/README.md
