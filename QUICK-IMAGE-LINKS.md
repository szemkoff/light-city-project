## How to Use These Links

1. **Click the link** for the technology you want
2. **Browse images** and select high-quality photos
3. **Download** (look for "Free Download" button)
4. **Save to**: `/Users/stantheman/Light City Project/website/static/img/technology/`
5. **Name file**: `technology-name-description.jpg` (e.g., `clt-construction-building.jpg`)
6. **Optimize**: Use [TinyPNG](https://tinypng.com) if file >500KB
