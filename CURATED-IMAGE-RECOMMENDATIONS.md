## üì• HOW TO DOWNLOAD & USE

### Step 1: Search & Select
1. Click the links above or search manually on the sites
2. Look for high-resolution images (1920px+ width)
3. Check the license (all these sites offer free commercial use)

### Step 2: Download
- **Unsplash**: Click "Download free" button
- **Pexels**: Click "Free Download" and select size
- **Pixabay**: Click "Free Download" and choose resolution
- **Wikimedia Commons**: Click image ‚Üí "Download" ‚Üí Select size

### Step 3: Optimize
1. Rename file descriptively (e.g., `crystal-energy-field.jpg`)
2. Compress if needed: https://tinypng.com/
3. Resize if too large: Max 1200px width recommended

### Step 4: Add to Project
```bash
# Place in appropriate folder:
website/static/img/concepts/       # Concept photos
website/static/img/energy/         # Energy/crystal images
website/static/img/meditation/     # Consciousness images
website/static/img/architecture/   # Building photos
```

### Step 5: Reference in Markdown
```markdown
![Crystal Energy Field](/img/energy/crystal-energy-field.jpg)

*Natural quartz crystal displaying the geometric precision found in Light City's energy architecture.*
```
