# Diagrams

This folder houses canonical diagrams for Light City.

## Structure
- `diagram-*.png|svg` – exported diagrams
- `src/` – editable sources (Figma, Blender, Sketch, etc.)

## Current Needed Diagrams
- City Plan (top-down)
- Energy Grid topology
- Central Spire cross-section
- Communosphere interior concept
- Sacred geometry overlays

## Style Cheatsheet
- Lines: Deep Space Blue (#0B2447) @ 70% opacity
- Highlights: Celestial Blue (#4A9EFF)
- Energy: Solar Gold (#E6B400) → White Gold (#FFF6E0)
- Labels: Small caps, muted gray, 11–12px

## Versioning
Use `-v1`, `-v2` suffixes; keep sources in `src/`.
