# Technology Catalog - Image Sourcing Guide

This guide provides free, high-quality image sources for technologies featured in the Technology Catalog.

## Image Sources (Free/Open License)

### Best Sources for Technology Photos:
1. **Unsplash** (https://unsplash.com) - Free, high-quality photos
2. **Pexels** (https://pexels.com) - Free stock photos
3. **Wikimedia Commons** (https://commons.wikimedia.org) - Open license images
4. **Pixabay** (https://pixabay.com) - Free images and videos
5. **Manufacturer websites** - Often provide press/media kits

## Image Specifications

**Recommended Sizes**:
- Width: 1200-1600px (will be responsive)
- Format: JPG (photos), PNG (diagrams with transparency)
- File size: <500KB (optimize with TinyPNG or similar)

**Naming Convention**:
```
technology-name-description.jpg

Examples:
clt-construction-building.jpg
geodesic-dome-structure.jpg
vertical-farm-aerofarms.jpg
bipv-solar-facade.jpg
```

**Storage Location**:
```

```

## How to Add Images to Technology Catalog

### 1. Download Images
- Use free sources listed above
- Ensure proper licensing (CC0, Public Domain, or with attribution)
- Optimize file size (<500KB)

### 2. Save to Project
```bash
mkdir -p "/Users/stantheman/Light City Project/website/static/img/technology"
# Save images to this folder
```

### 3. Add to Markdown
In `technology-catalog.md`, add after each technology description:

```markdown
### Cross-Laminated Timber (CLT)

**Status**: ✅ Commercially Available (TRL 8-9)  
**Cost**: $$ (Medium)

![CLT Construction](/img/technology/clt-construction-building.jpg)
*Modern CLT construction - Mjøstårnet, Norway (85.4m tall, 18 stories)*

**Description**:
...
```
