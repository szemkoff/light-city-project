## USAGE INSTRUCTIONS

1. Copy the prompt for the image you want to generate
2. Go to FluxFast 1.1 (https://fluxfast.ai) or your preferred tool
3. Paste the prompt
4. Set the generation settings (size, steps, etc.)
5. Generate 4 variations
6. Select the best one
7. Download and optimize (convert to WebP, create responsive sizes)
8. Add to /website/static/img/homepage/
9. Update the tracking table in .cursorignore-image-generation-guide.md

## OPTIMIZATION COMMANDS

# Convert to WebP
cwebp -q 85 original.jpg -o optimized.webp

# Create responsive sizes
convert original.jpg -resize 400x optimized-400w.jpg
convert original.jpg -resize 800x optimized-800w.jpg
convert original.jpg -resize 1200x optimized-1200w.jpg
convert original.jpg -resize 1600x optimized-1600w.jpg

# Convert all to WebP
for file in optimized-*.jpg; do
  cwebp -q 85 "$file" -o "${file%.jpg}.webp"
done
