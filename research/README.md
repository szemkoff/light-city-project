Use clear, searchable names:
- `2025-10-10-sacred-architecture-study.md`
- `findhorn-community-governance-notes.md`
- `crystalline-structures-research.pdf`

### Note-Taking Format

For research notes, include:
```markdown
# Source Title

**Date**: When accessed/read
**Type**: Book/Article/Video/etc.
**Author**: Creator name
**Link**: URL if applicable

## Key Points
- Bullet points of main ideas

## Relevant Quotes
> Direct quotes with page numbers

## Applications to Light City
How this research informs the project

## Further Research
Questions or areas to explore more

---
**Tags**: #architecture #sacred-geometry #community
```
